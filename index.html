<h1> a <h6>

As tags <h1> a <h6> são usadas para definir títulos HTML.

<h1> define o título mais importante. <h6> define o título menos importante.

Nota: Use apenas um <h1> por página - isso deve representar o título/assunto principal para a página inteira. Além disso, não pule os níveis de título - comece com <h1>, depois use <h2>, e assim por diante.

<p>

A <p> tag define um parágrafo.

Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada <p> elemento.

Dica: use CSS para estilizar parágrafos.

<mark>

A <mark> tag define o texto que deve ser marcado ou destacado.

Exemplo: Frase destacada para evidenciar.

<small>

A <small> tag define textos menores (como direitos autorais e outros comentários pararelos).

Dica: Esta tag não está obsoleta, mas é possível obter um efeito mais rico (ou o mesmo) com CSS.

<i>

A <i> tag define uma parte do texto em uma voz ou humor alternativo. O conteúdo dentro é normalmente exibido em itálico .

A <i> tag é frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Use o <i> elemento somente quando não houver um elemento semântico mais apropriado, como:

<em> (texto enfatizado)
<strong> (texto importante)
<mark> (texto marcado/destacado)
<cite> (o título de uma obra)
<dfn> (um termo de definição)

<u>

A <u> tag representa algum texto que não é articulado e estilizado de forma diferente do texto normal, como palavras com erros de ortografia ou nomes próprios em texto chinês. O conteúdo dentro é normalmente exibido com um sublinhado. Você pode alterar isso com CSS (veja o exemplo abaixo).

Dica: Evite usar o <u> elemento onde ele possa ser confundido com um hiperlink!

<strong>

A <strong> tag é usada para definir texto com forte importância. O conteúdo dentro é normalmente exibido em negrito.

Dica: Use a tag <b> para especificar texto em negrito sem nenhuma importância extra!

<ol> <ul> <li>

  • <ol>

    A <ol> tag define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

  • <ul>

    A <ol> tag define uma lista não ordenada (com marcadores).
    Use a <ol> tag junto com a tag

  • para criar listas não ordenadas.

  • <li>

    A <li> tag define um item da lista.
    A <li> tag é usada dentro de listas ordenadas ( <ol> ), listas não ordenadas ( <ul> ) e em listas de menu ( <menu> ).
    Em <ul> e <menu>, os itens da lista geralmente serão exibidos com marcadores.
    Em <ol>, os itens da lista geralmente serão exibidos com números ou letras.

<a>

A <a> tag define um hiperlink, que é usado para vincular de uma página a outra.

O atributo mais importante do elemento <a> é o atributo href, que indica o destino do link.

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

  • Um link não visitado é sublinhado e azul

  • Um link visitado é sublinhado e roxo

  • um link ativo é sublinhadoe vermelho

<hr>

A tag define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).<hr>

O elemento é mais frequentemente exibido como uma regra horizontal que é usada para separar o conteúdo (ou definir uma alteração) em uma página HTML. <hr>

<sub>

A tag define o texto subscrito. O texto subscrito aparece meio caractere abaixo do linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H<sub> 2O.

Ponta: Use a tag <sup> para definir texto sobrescrito.

<blockquote>

A tag <blockquote> em HTML é usada para indicar uma citação em bloco, ou seja, um trecho de texto retirado de outra fonte e exibido de forma destacada. Geralmente, os navegadores aplicam um recuo ao conteúdo dentro da <blockquote>, diferenciando visualmente a citação do restante do texto.

Exemplo:
<blockquote cite="https://www.exemplo.com/artigo">
"Este é um trecho citado de uma fonte externa."
</blockquote>

<font>

A tag <font> em HTML era usada para definir a fonte, cor e tamanho do texto, mas foi descontinuada no HTML5 e não deve mais ser utilizada. No lugar dela, recomenda-se o uso de CSS para estilizar textos.

<del>

A tag <del> define o texto que foi excluído de um documento. Os navegadores geralmente riscam uma linha no texto excluído.

Exemplo: <del>Texto excluído</del>
Saída: Texto excluído

<abbr>

A tag <abbr> define uma abreviação ou um acrônimo, como "HTML". "CSS", "Sr.", "Dr.", "ASAP", "ATM".

Exemplo: <abbr title="Associação Brasileira de Normas Técnicas">ABNT</abbr>

Saída: ABNT
